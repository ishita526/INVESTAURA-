# INVESTAURA

INVESTAURA is a trading website where users can **learn trading from scratch** with simulations.  
This project focuses on the **frontend development** of the platform.

## Features
- Interactive learning experience for beginners in trading.
- Simulated trading environment to practice strategies.
- User-friendly interface with dashboards and course pages.
- Responsive design for web use.

## Tech Stack
- HTML, CSS, JavaScript
- Firebase (for future backend integration)

## Project Status
- Frontend design completed.
- Backend integration and simulations planned for the next phase.
